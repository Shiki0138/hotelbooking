## 🔧 技術スタック
- Frontend: React + TypeScript + Vite
- Backend: Vercel Edge Functions
- Database: Supabase
- Auth: Supabase Auth
- Notification: SendGrid / Resend
- Monitoring: Vercel Analytics
- Payment: Stripe（将来）
